# DOM Racer — How to Install (Beta Testing)

Thanks for testing! Here's how to get DOM Racer running in 2 minutes.

## Step 1: Download

Download the `dom-racer.zip` file I sent you.

## Step 2: Unzip

Unzip the file into a folder. Remember where you put it — Chrome needs to read from that folder.

## Step 3: Load in Chrome (or Edge)

1. Open Chrome and go to `chrome://extensions` (or `edge://extensions` for Edge)
2. Turn on **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the **unzipped folder** (the one containing `manifest.json`)
5. Done! You should see "DOM Racer" appear in your extensions list

## Step 4: Play!

1. Go to any webpage (Wikipedia, GitHub, a blog, news site — anything with content)
2. Press **Shift + Space** to start the game
3. Drive with **WASD** or **arrow keys**
4. Press **Space** for nitro boost
5. Collect gold coins, dodge police, survive as long as you can
6. Press **Esc** to quit

## Quick Controls

| Key | Action |
|-----|--------|
| Shift + Space | Start / stop the game |
| WASD / Arrows | Drive |
| Space | Nitro boost (3s cooldown) |
| V | Switch car (coupe / buggy / truck — each drives differently!) |
| M | Toggle music |
| R | Restart run |
| Esc | Quit |

## Tips

- **Every page is different** — try it on different websites for unique tracks
- **The buggy** turns fast but is slower. **The truck** is fast but turns like a boat.
- **Watch for the airplane** — it drops bonus coins and power-ups
- **After ~35 seconds**, bushes start growing from walls and close off your routes
- The game works best on content-heavy pages. If a page doesn't have enough elements, you'll see a friendly "not enough to race on" screen.

## Troubleshooting

- **Game won't start?** Make sure you're on a regular webpage (not `chrome://` pages or the new tab page)
- **Shift+Space conflicts with the page?** Try **Shift + Backtick** (`` ` ``) as an alternative, or use the extension popup button
- **No coins on the page?** Some heavily-iframe'd pages (Gmail, Figma) don't have enough scannable elements
